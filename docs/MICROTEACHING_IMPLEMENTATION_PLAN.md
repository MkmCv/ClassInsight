## 📊 评估指标设计（基于实际场景）

### 核心指标

1. **PPT依赖度**（0-100分）
   - `screen` 使用时间占比
   - 建议：60-70%为优秀，>80%为过度依赖

2. **板书技能**（0-100分）
   - `blackboard-writing` 使用情况
   - 建议：10-20%为优秀，0%为不足

3. **互动意识**（0-100分）
   - `guide`、`answer`、`On-stage interaction` 使用情况
   - 建议：有互动为优秀，无互动为不足

4. **教学方式多样性**（0-100分）
   - 使用的教学方式种类
   - 建议：3种以上为优秀，1-2种为不足

5. **时间分配合理性**（0-100分）
   - 各教学方式时间占比的合理性
   - 建议：平衡分配为优秀，严重失衡为不足

### 综合得分计算

```
综合得分 = (
    PPT依赖度得分 × 0.25 +
    板书技能得分 × 0.25 +
    互动意识得分 × 0.20 +
    教学方式多样性得分 × 0.15 +
    时间分配合理性得分 × 0.15
)
```

## 🎯 建议的改进方向（基于实际场景）

### 高优先级建议

1. **减少PPT依赖**
   - 问题：过度依赖PPT（>80%）
   - 建议：在重点内容时使用白板板书补充
   - 目标：PPT使用时间降至60-70%

2. **增加板书使用**
   - 问题：完全不使用白板
   - 建议：在关键知识点时使用白板板书
   - 目标：板书时间占比达到10-20%

3. **增强互动意识**
   - 问题：没有模拟互动
   - 建议：增加模拟提问、回答、邀请上台等互动
   - 目标：每5-10分钟有一次互动

### 中优先级建议

4. **优化时间分配**
   - 问题：时间分配失衡
   - 建议：平衡各教学方式的时间占比
   - 目标：讲授50% + 板书15% + 互动15% + PPT20%

5. **增加手势指导**
   - 问题：手势指导不足
   - 建议：增加面向镜头的手势指导
   - 目标：体现互动意识

### 低优先级建议

6. **优化教学节奏**
   - 问题：节奏单一
   - 建议：增加节奏变化
   - 目标：快慢结合，重点突出
