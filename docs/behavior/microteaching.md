# 微格课堂分析方案

## 📚 场景说明

**微格课堂（Microteaching）**：师范生面对镜头录制自己上课的视频，用于教学技能训练和评估。

### 实际场景特点（基于真实微格课堂视频）
- **单人场景**：师范生独自站在教室前面，面对大屏幕（如 seewo 交互式显示屏）和镜头
- **多媒体教学**：使用 PPT/交互式屏幕展示教学内容
- **模拟教学**：需要展示完整的教学流程和技能，虽然没有真实学生，但要模拟有学生的场景
- **设备使用**：通常手持遥控器/点击器控制PPT，旁边可能有白板
- **技能评估**：重点评估教学技能、互动设计、教学方式、多媒体使用等
- **自我反思**：通过视频回看和改进建议，提升教学能力

### 典型场景元素
- ✅ **大屏幕/交互式显示屏**：显示PPT教学内容（可检测：`screen`）
- ✅ **白板**：可能用于板书补充（可检测：`blackBoard`）
- ✅ **师范生站立**：保持教师姿态（可检测：`stand`）
- ✅ **讲授行为**：讲解知识点（可检测：`teacher`）
- ✅ **手势指导**：使用手势辅助讲解（可检测：`guide`）
- ✅ **板书**：可能使用白板进行板书（可检测：`blackboard-writing`）

## 🎯 可检测的教学技能指标

基于现有的 `comprehensive_scene_best.pt` 模型，可以检测以下指标：

### 1. **教学方式多样性** ⭐⭐⭐⭐⭐

**检测指标**（基于实际微格课堂场景）：
- `teacher`（讲授）：师范生讲解知识点的时间占比
- `screen`（屏幕/多媒体）：PPT/交互式屏幕使用情况（**微格课堂常见**）
- `blackboard-writing`（板书）：白板板书使用频率和时长（**加分项**）
- `guide`（指导手势）：手势指导的使用（**体现互动意识**）

**实际场景分析**：
- **典型场景**：师范生面对大屏幕，使用PPT教学，手持遥控器
- **检测重点**：是否只依赖PPT，还是有板书补充
- **评估维度**：
  - **多媒体依赖度**：`screen` 使用时间占比（不应过高）
  - **板书补充**：`blackboard-writing` 使用情况（体现教学技能）
  - **手势辅助**：`guide` 使用频率（体现互动意识）

**建议方向**：
```
✅ 优秀：PPT讲解 + 白板板书补充 + 手势指导（3种以上）
⚠️ 改进：只有PPT讲解，建议增加白板板书或手势指导
❌ 不足：单一依赖PPT，建议学习板书技能和互动技巧
```

---

### 2. **互动设计能力** ⭐⭐⭐⭐⭐

**检测指标**（微格课堂特殊场景）：
- `answer`（应答）：模拟学生回答问题的互动（**需要师范生自己模拟**）
- `On-stage interaction`（台上互动）：模拟邀请"学生"上台的互动设计
- `guide`（指导手势）：面向"学生"的手势指导（**可检测**）

**实际场景分析**：
- **挑战**：微格课堂没有真实学生，互动需要师范生自己模拟
- **检测重点**：
  - 是否有模拟提问和回答的行为（`answer`）
  - 是否有面向镜头的互动手势（`guide`）
  - 是否有模拟学生上台的设计（`On-stage interaction`）

**评估维度**：
- **互动意识**：是否有模拟互动的行为
- **互动频率**：整节课的模拟互动次数
- **互动自然度**：互动是否自然流畅（需要人工评估）

**建议方向**：
```
✅ 优秀：有模拟提问、回答、邀请上台等互动设计
⚠️ 改进：互动较少，建议增加模拟互动环节
❌ 不足：完全没有互动，建议学习互动设计技巧
💡 提示：即使没有真实学生，也要模拟互动，体现互动意识
```

---

### 3. **板书技能** ⭐⭐⭐⭐⭐（微格课堂重要指标）

**检测指标**：
- `blackboard-writing`（板书）：白板板书行为检测
- `blackBoard`（白板）：白板使用情况
- 板书时长和频率

**实际场景分析**：
- **典型场景**：教室有白板，但很多师范生只使用PPT，不使用白板
- **检测重点**：
  - 是否使用白板进行板书补充（`blackboard-writing`）
  - 板书与PPT的配合使用
  - 板书时机是否恰当

**评估维度**：
- **板书使用率**：板书时间占总时长比例（建议10-20%）
- **板书时机**：是否在关键知识点使用板书
- **板书与PPT配合**：板书是否作为PPT的补充和强调
- **板书质量**：板书是否清晰、有条理（需要人工评估）

**建议方向**：
```
✅ 优秀：PPT + 白板板书补充，板书清晰、有条理，时机恰当
⚠️ 改进：只有PPT，建议在重点内容时使用白板板书补充
❌ 不足：完全不使用白板，建议学习板书设计技巧
💡 提示：白板板书可以突出重点，比PPT更有针对性
```

---

### 4. **多媒体使用能力** ⭐⭐⭐⭐（微格课堂基础技能）

**检测指标**：
- `screen`（屏幕）：PPT/交互式屏幕使用情况（**微格课堂必备**）
- 屏幕使用时长和频率
- 屏幕与讲授的配合

**实际场景分析**：
- **典型场景**：师范生面对大屏幕（如 seewo），使用PPT教学
- **检测重点**：
  - PPT使用是否合理（不应过度依赖）
  - PPT与讲授的配合是否自然
  - 是否只是"读PPT"还是"讲PPT"

**评估维度**：
- **多媒体使用率**：PPT使用时间占比（建议60-70%，不应过高）
- **多媒体与讲授配合**：是否只是读PPT，还是有讲解和补充
- **多媒体切换时机**：PPT切换是否自然，是否有停顿讲解

**建议方向**：
```
✅ 优秀：PPT作为辅助，配合讲解、板书、互动
⚠️ 改进：过度依赖PPT，建议增加板书和互动环节
❌ 不足：只是读PPT，建议学习如何讲解PPT内容
💡 提示：PPT是工具，不是全部，要配合其他教学方式
```

---

### 5. **教学姿态和仪态** ⭐⭐⭐

**检测指标**：
- `stand`（站立）：是否保持站立姿态
- 站立时长和频率
- 姿态稳定性

**评估维度**：
- **站立时间**：是否保持教师应有的站立姿态
- **姿态变化**：是否有适当的位置移动
- **仪态表现**：是否展现出教师的专业形象

**建议方向**：
```
✅ 优秀：保持站立，姿态自然，有适当移动
⚠️ 改进：站立时间不足，建议保持教师姿态
❌ 不足：长时间坐着，建议学习教师仪态规范
```

---

### 6. **时间分配合理性** ⭐⭐⭐⭐（微格课堂关键指标）

**检测指标**：
- 各教学方式的时间占比
- 讲授、互动、板书、多媒体的时间分配
- 时间分布是否均匀

**实际场景分析**（基于微格课堂）：
- **典型问题**：很多师范生过度依赖PPT，讲授时间过长
- **检测重点**：
  - PPT使用时间占比（不应超过70%）
  - 板书使用时间占比（建议10-20%）
  - 互动设计时间占比（建议10-15%）

**评估维度**：
- **讲授时间**：是否过长或过短（建议50-60%）
- **PPT时间**：是否过度依赖（建议60-70%，不应更高）
- **板书时间**：是否合理（建议10-20%）
- **互动时间**：是否足够（建议10-15%）
- **时间平衡**：各环节时间分配是否合理

**建议方向**（微格课堂标准）：
```
✅ 优秀：PPT讲解50% + 板书15% + 互动15% + 讲授20%
⚠️ 改进：PPT时间过长（>80%），建议增加板书和互动
❌ 不足：只有PPT讲解，建议学习时间分配技巧
💡 提示：微格课堂要展示多种教学技能，不能只依赖PPT
```

---

### 7. **教学流程完整性** ⭐⭐⭐⭐

**检测指标**：
- 是否包含：导入 → 讲授 → 互动 → 总结
- 各环节的时长和顺序
- 环节转换是否自然

**评估维度**：
- **流程完整性**：是否包含完整的教学环节
- **环节顺序**：是否符合教学逻辑
- **环节转换**：转换是否自然流畅

**建议方向**：
```
✅ 优秀：包含完整的教学流程，环节转换自然
⚠️ 改进：缺少某些环节，建议完善教学流程
❌ 不足：流程不完整，建议学习标准教学流程
```

## 🔍 需要新增的检测能力

### 1. **语音分析**（重要，建议优先实现）
- **语速**：是否过快或过慢（微格课堂常见问题）
- **停顿**：是否有适当的停顿（给"学生"思考时间）
- **音量**：是否清晰可听
- **语调**：是否有适当的语调变化（避免单调）
- **口头禅**：是否有过多的"嗯"、"啊"等口头禅

### 2. **眼神和表情**（重要，微格课堂关键）
- **眼神交流**：是否面向镜头（模拟与学生眼神交流）
- **表情管理**：是否保持适当的教学表情（微笑、专注等）
- **面部朝向**：是否面向镜头（学生），而不是只看着屏幕
- **眼神方向**：是否在屏幕和镜头之间切换（模拟与学生互动）

### 3. **手势和动作**（部分支持，可优化）
- **手势指导**：`guide` 可以检测，但可以更细化
- **动作幅度**：是否有适当的手势动作（不要太小或太大）
- **位置移动**：是否有适当的位置变化（不要一直站在一个地方）
- **遥控器使用**：是否自然使用遥控器/点击器

### 4. **教学节奏**（可计算）
- **快慢节奏**：基于行为转换频率
- **重点突出**：基于板书和互动的时机
- **节奏变化**：是否有适当的节奏变化
- **PPT切换频率**：切换是否过快或过慢

### 5. **面向镜头意识**（微格课堂特有）
- **镜头朝向**：是否面向镜头（模拟面向学生）
- **镜头与屏幕切换**：是否在屏幕和镜头之间自然切换
- **互动模拟**：是否有面向镜头的互动行为
