## 📋 页面组织建议

### 当前组织方式
```
pages/
├── 1_🏠_首页.py          (所有用户)
├── 2_📤_视频上传.py       (所有用户)
├── 3_📈_行为分析.py       (所有用户)
├── 4_💡_教学建议.py       (所有用户)
├── 5_👤_用户管理.py       (仅管理员)
└── 6_📅_课表管理.py       (仅管理员)
```

### 建议的组织方式
```
pages/
├── 1_🏠_首页.py                    (所有用户)
├── 2_📤_视频上传.py               (所有用户)
├── 3_📈_行为分析.py               (所有用户)
├── 4_💡_教学建议.py               (所有用户)
└── Admin/                          (管理员专用目录)
    ├── 0_👑_管理员中心.py          (管理员仪表盘)
    ├── 1_👤_用户管理.py            (重命名)
    └── 2_📅_课表管理.py            (重命名)
```

**注意**: Streamlit 不支持子目录，所以需要使用命名约定。

### 推荐的命名方式
```
pages/
├── 1_🏠_首页.py
├── 2_📤_视频上传.py
├── 3_📈_行为分析.py
├── 4_💡_教学建议.py
├── Admin_0_👑_管理员中心.py        (管理员仪表盘)
├── Admin_1_👤_用户管理.py          (重命名)
└── Admin_2_📅_课表管理.py          (重命名)
```

这样在Streamlit侧边栏中，管理员页面会集中显示在一起。
