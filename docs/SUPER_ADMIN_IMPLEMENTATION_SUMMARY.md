## 🔐 权限层级

### 三级权限结构

1. **普通教师** (`teacher`)
   - 访问：首页、视频上传、行为分析、教学建议
   - 系统：`System/frontend/app.py`

2. **管理员** (`admin`)
   - 访问：所有教师功能 + 用户管理、课表管理、登录失败摘要
   - 系统：`System/frontend/app.py`
   - 不能：修改登录安全配置、管理模型

3. **超级管理员** (`super_admin`)
   - 访问：独立的超级管理员系统
   - 系统：`System/super_admin/app.py`（独立启动）
   - 功能：登录安全配置、模型管理

## 🚀 使用方式

### 启动超级管理员系统

```bash
# 进入超级管理员目录
cd "H:\毕业设计\System\System\super_admin"

# 启动 Streamlit 应用（使用不同端口避免冲突）
streamlit run app.py --server.port 8502
```

### 访问地址

- **普通系统**：`http://localhost:8501`（默认）
- **超级管理员系统**：`http://localhost:8502`

### 创建超级管理员账号

需要在数据库中创建，或通过后端 API：

```python
# 示例代码（需要在后端执行）
from app.core.security import get_password_hash
from app.models.user import User, UserRole

super_admin = User(
    username="superadmin",
    email="superadmin@example.com",
    password_hash=get_password_hash("your_password"),
    role=UserRole.SUPER_ADMIN.value,
    is_active=1
)
```
