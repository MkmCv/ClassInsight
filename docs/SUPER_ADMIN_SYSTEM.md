## 🔐 权限控制

### 角色层级

1. **普通教师** (`teacher`)
   - 可以访问：首页、视频上传、行为分析、教学建议
   - 不能访问：管理员页面、超级管理员系统

2. **管理员** (`admin`)
   - 可以访问：所有教师功能 + 用户管理、课表管理、登录失败摘要
   - 不能访问：登录安全配置修改、模型管理、超级管理员系统

3. **超级管理员** (`super_admin`)
   - 可以访问：独立的超级管理员系统
   - 功能：登录安全配置、模型管理

### 登录方式

- **普通系统**：`System/frontend/app.py` - 教师和管理员登录
- **超级管理员系统**：`System/super_admin/app.py` - 仅超级管理员登录

## 📋 使用说明

### 创建超级管理员账号

需要在数据库中手动创建或通过后端 API 创建：

```python
# 示例：创建超级管理员账号
from app.core.security import get_password_hash
from app.models.user import User, UserRole

super_admin = User(
    username="superadmin",
    email="superadmin@example.com",
    password_hash=get_password_hash("your_password"),
    role=UserRole.SUPER_ADMIN.value,
    is_active=1
)
```

### 访问超级管理员系统

1. **启动超级管理员系统**：
   ```bash
   cd "H:\毕业设计\System\System\super_admin"
   streamlit run app.py --server.port 8502
   ```

2. **访问地址**：`http://localhost:8502`

3. **登录**：使用超级管理员账号登录

### 功能使用

#### 登录安全配置
1. 查看当前配置
2. 修改配置值（表单输入）
3. 系统会提供修改代码文件的详细说明
4. 按照说明修改 `backend/app/core/config.py`
5. 重启后端服务

#### 模型管理
1. 查看所有模型文件
2. 查看模型信息（大小、修改时间）
3. 更新模型（开发中）
4. 激活模型（开发中）
